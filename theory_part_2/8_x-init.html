One level up

x-init See lecture on GitHub

Objectives:
  1. Move the data from the component into separate script.
  2. Use x-init to create 2 instances of the tab component with different data on one page.

Instructions

Move Data into seperate file

  1. Create a file tabData.js and include it into your HTML.
  2. Move the data of your tabs there by declaring the myTabs array as a variable.
  3. Test if everything still works.

Second Tabs on same page

  1. Copy the section with the tabs and paste it underneath the first tabs.
  2. Duplicate the array with the data in the external js file and give it a different name and different data.
  3. Now initialize each of the tab instances with different data by using x-init.
    E.g.: x-init="myTabs = cat2"

Refreshing of loaded data

  • To make sure that every instance of your component keeps working with the correct data, you have to reassign the data upon the users action like so:
    @click="tab=index; myTabs = cat_2";
