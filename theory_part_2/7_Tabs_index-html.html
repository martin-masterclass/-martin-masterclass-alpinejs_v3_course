Instructions

  1. Copy the section with the tab example and place it under the other tabs. Test if the second tabs are working.
  2. Copy the array from the file 7_data.txt into the function dynamicTabs() at the bottom of the document.
  3. Refactor the second example: Make x-for loop each for the tab headers and for the tab content. That means that you need 2 template tags, with each one element inside.
  4. In both templates you loop over myTabs. In the first one you output the title and in the second the body.
  5. Please be careful that you do not use "tab" as the loop variable, because this is already used for the active tab. It should be like this: x-for="myTab in myTabs".
  6. Change x-for="myTab in myTabs" into x-for="(myTab, index) in myTabs", so you have access to the index (=number) of the tab.
  7. Now change everything that has to do with the number of the tab into index, e.g. @click="tab=index"
  8. Use x-html to render the HTML in the tab bodies.
  9. Now the tabs should work with the dynamic content.
  10. Just for fun, try to add another tab, by adding it to the data.
