Example 2: Show element only with condition x-if

If you want to use x-if you have to wrap the element, that you want to show or hide with a template-tag.

In this case the element will be removed completely from the DOM on hide.

Steps Example 1

  1. Give the section an object x-data="{show : false}"
  2. Make the visibility of the span dependent on the object show by giving it a x-show="show", and try it out.
  3. Give the button a @click="show = !show;", to toggle the visibility on click.
  4. Change the text on the button dependent on show, by giving it a x-text="!show ? 'Show' : 'Hide'"
  5. Extract the basic button classes (without the color) into a second property of your "section-object": buttonClass: 'transition p-1 rounded shadow focus:outline-none ', don't forget a space at the end!
  6. Make the button color dynamic and dependent on show, by concatenating different color classes with the help of a ternary expression like this:
    :class="!show ? buttonClass + 'bg-green-300 hover:bg-green-400' : buttonClass + 'bg-red-300 hover:bg-red-400'"
  7. Test everything and open Chrome DevTools. In DevTools observe how the span gets a style of display: none, when you click on the Hide.

Steps Example 2

  1. Copy the complete Example 1 into Example 2.
  2. Wrap the span that you want to show/ hide with a Template-Tag
  3. Give the starting template-tag x-if="show".
  4. Test the example and open Chrome DevTools.
  5. In DevTools observe how the complete HTML appears/ disappears on Show/ Hide.
