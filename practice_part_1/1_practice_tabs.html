Instructions

Change Tab content upon click on Tab

  1. Give the surrounding section a x-data="{tab : 1}", to determine the default active tab.
  2. Remove the display:none from tab2Content and tab3Content.
  3. Make all 3 tab contents depend on the key "tab", by using x-show, e.g. in tab1Content:
    x-show="tab == 2"
  4. Test, if the tab changes upon changing the object: tab.
  5. Give each "Tab span" an @click to change the object "tab", e.g. @click="tab=1"
  6. Test if you can change the tab content with clicking on the tabs.

Change the active tab to darker color without hover

  1. You need 3 more properties in your object: One for the tab base styles, eg. tabClasses, tabNormalColors and tabHiColor.
  2. Extract the tab base classes into tabClasses. Don't forget a space at the end.
  3. Extract the normal tab color classes (together with the hover class) into tabNormalColors.
  4. Extract the tab higlight color class into tabHiColor.
  5. After extracting remove all remaining classes from the tabs and bind the class attributes of the 3 Tabs with a colon (:).
  6. Concatenate in the class attibutes of the normal tabs the tabClasses with either tabNormalColors, or with tabHiColor if the current tab is selected.
  7. Use a ternary expression for this, e.g.:
    :class="tab == 1 ? tabClasses + tabHiColor : tabClasses + tabNormalColors"
  8. Test if everything works.
