Alpine.js challenge: Color Slider See lecture on GitHub

Objectives:
  1. Output of decimal values when one of the sliders is moved.
  2. On the fly concatenation to RGB color value with display of the value and application in div. The dynamic result will be e.g.
    background-color='#A033AC'

In this example you can create
255 x 255 x 255 = 16.581.375 different colors.

Todos:

  1. Fix problem for one digit Hex values
  2. Make it possible to enter Hex values manually
  3. Avoid "NAN problem" when an input is made empty

Instructions

  1. Give the section an x-data with the 3 properties red, green and blue, and give each of them a default of '128' (as a string).
  2. Give the 3 slider inputs AND also the inputs for the decimal value each a x-model with the according color, e.g.: x-model="red".
    Now you can already see the value of '128' in each input. You can move each slider and see how the value changes and vice versa you can also change the number in the decimal input to move the slider.
  3. Now you bind the value of each "Hex Input" with e.g. :value="parseInt(red).toString(16).toUpperCase()"
    Now you should see the according hexadecimal value in the input fields.
  4. Bind the result div (id="result") to:
    :style="'background-color:#' + parseInt(red).toString(16) + parseInt(green).toString(16) + parseInt(blue).toString(16)"
